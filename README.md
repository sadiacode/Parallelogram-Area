# Parallelogram Area
## Description
This project is a simple python program that calculates the area of parallelogram.
## Technologies Used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. Enter the base and height.
5. View the result of calculation.
## Sample Output
```Area
Enter base: 2
Enter height: 3

Area of parallelogram is: 6.0
```
